Help

Installation

This product includes HTML (hypertext markup language) files for the navigation pages.

If you have an up-to-date version of a browser to read HTML pages (for example, Internet Explorer®, Firefox®, Safari® or Google Chrome®), you are ready to access the contents of the product.

Once you have the required software installed, you can now open the file "index.html" in your browser to start navigating the product.

Usage tip: When you access a QuestionBank, it will open in a new browser window

Installation to a web server

The product can be made available on an intranet web server by moving the contents into the server file space, and providing users with a link to the "index.html" file. The product is self-contained.
